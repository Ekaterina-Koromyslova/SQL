**Структура репозитория**
1 папка = 1 проект
Внутри каждой папки есть подробное описание заданий. Код находится по пути src/ex_/day_ex_.sql
- Индивидуальные проекты имеют название DAY_
- Групповые проекты имют название TEAM_

**Описание проекта**
SQL Bootcamp — это комплексный учебный курс, в рамках которого последовательно изучаются ключевые аспекты работы с реляционными базами данных. В ходе выполнения заданий я прошла путь от базовых выборок и соединений таблиц до построения рекурсивных запросов, разработки триггеров и функций, моделирования графов и решения задач оптимизации, а также работы с архитектурой хранилищ данных (DWH). Каждое упражнение приближено к реальным кейсам аналитики, разработки бизнес-логики и работы с транзакционными системами.

Курс состоит из последовательных тематических дней, каждый из которых формирует новое направление компетенций: агрегирование данных, создание представлений, оптимизация через индексы, управление транзакциями и уровнями изоляции, построение ETL-логики, аудит изменений, вычисления внутри СУБД, работа с графами и маршрутизацией, анализ “грязных” данных и формирование отчётности.
